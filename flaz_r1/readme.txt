Программа для "внутреннего" обмена папками удаленных офисов, без удаленного доступа к RDP центральным серверам.
Папка копируется в архив, архив по протоклу FTP пересылается на сервер.
К имени пользователя "привязывается" порт подключения, для нестандартных портов обмена по протоколу FTP.
Пароль FTP составляется по простому склеиванию нескольких констант, и в настройках программы не учавствует.
Для "разворачивания" программы на рабместе необходим обмен самим EXEшником.

7z.exe - подразумевается что этот файл будет размещаться в папке с компилируемым проектом
